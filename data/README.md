# Data Directory

This directory contains both **input files** (that you provide) and **output files** (generated by the pipeline). **Nothing under `data/` is committed to git** except this README — users provide their own copies of the raw OpenSNP files.

---

## 📥 Required Input Files

Place these files in the `data/` directory before running the pipeline:

| Path | Description | Rows/Files | Source |
|------|-------------|------------|--------|
| `opensnp_Ancestry.csv` | Self-reported ancestry strings | 414 rows | Class OpenSNP browser |
| `opensnp_genotypes_Ancestry__413files/manifest.csv` | Genotype filename → user_id mapping | 405 rows | Class data package |
| `opensnp_genotypes_Ancestry__413files/*.txt` | Raw genotype files (23andMe/Ancestry format) | 405 files | OpenSNP downloads |

### File Format Examples

**opensnp_Ancestry.csv:**
```csv
user_id,value
1,British
2,German + Irish
3,East Asian
...
```

**manifest.csv:**
```csv
user_id,genotype_filename,genotype_format,chrom_sex
1,user_1_file1_23andme.txt,23andme,XY
2,user_2_file2_ancestry.txt,ancestry,XX
...
```

**Genotype file (23andMe format):**
```
# build 37
rsid	chromosome	position	genotype
rs3094315	1	742429	AG
rs12124819	1	766409	AA
...
```

```
data/
├── build_verification_results.csv             # Inferred genome builds

    ├── bed_inputs/user_*.grch36.bed
    ├── lifted_bed/user_*.grch37.bed
    └── unmapped/user_*.unmapped.bed
```

### Step 05: PLINK Conversion
```
data/
├── plink_individual/                      # One PLINK dataset per user
│   ├── user_1.bed                         # Binary genotype data
│   ├── user_1.bim                         # SNP information
│   ├── user_1.fam                         # Sample information
│   ├── user_1.log                         # PLINK log file
│   └── ...                                # (296 sets of files)
├── stage3_conversion_log.csv              # Successful conversions
├── stage3_failed_samples.csv              # Failed conversions + reasons
└── stage4_input_manifest.csv              # 296 samples for merging
```

### Step 06: Merge
```
data/
└── plink_merged/full_merge/strict_retry/
    ├── full_merged_strict.bed             # Merged binary genotypes
    ├── full_merged_strict.bim             # 93,476 shared SNPs
    ├── full_merged_strict.fam             # 296 samples
    ├── full_merged_strict.log             # Merge log
    ├── full_merged_strict-merge.missnp    # Excluded SNPs (strand issues)
    ├── filtered_inputs/                   # Individual files filtered to panel
    │   └── user_*_v1.{bed,bim,fam}
    ├── snp_panel_v1.txt                   # Shared SNP list (attempt 1)
    ├── final_filtering_log.csv            # Per-sample filtering results
    └── merged_sample_manifest.csv         # Sample metadata
```

### Step 07: QC, LD Pruning, IBS
```
data/
└── plink_qc/
    ├── step1_sample_filtered.{bed,bim,fam}           # After sample missingness filter
    ├── step2_snp_filtered.{bed,bim,fam}              # After SNP missingness filter
    ├── step3_maf_filtered.{bed,bim,fam}              # After MAF filter
    ├── step3b_no_ambiguous.{bed,bim,fam}             # After removing A/T, C/G SNPs
    ├── step4_pruned_dataset.{bed,bim,fam}            # Final: 260 samples, 53,131 SNPs
    ├── step5_ibs.genome                              # PLINK pairwise IBS (33,670 pairs)
    ├── step5_ibs_with_proportions.csv                # IBS + computed proportions
    ├── qc_stage_summary.csv                          # QC waterfall table
    ├── qc_missingness.{imiss,lmiss}                  # Missingness reports
    ├── samples_fail_missingness.txt                  # Excluded samples
    ├── ambiguous_snps.txt                            # A/T, C/G SNPs removed
    ├── step4_ld_prune.prune.{in,out}                 # LD pruning results
    ├── candidate_related_pairs_IBS2_gt_0.4.csv       # High IBS2 pairs
    └── candidate_related_pairs_PIHAT_gt_0.125.csv    # ≥3rd degree relatives
```

## 💾 Disk Space Requirements

Approximate disk usage:

| Directory | Size | Description |
|-----------|------|-------------|
| Input files | ~5-10 GB | Raw genotype files (405 files) |
| `plink_individual/` | ~2-3 GB | Individual PLINK binaries (296 samples) |
| `plink_merged/` | ~500 MB | Merged dataset + filtered copies |
| `plink_qc/` | ~300 MB | QC'd datasets + IBS results |
| Other intermediates | ~200 MB | CSV manifests, logs, etc. |
| **Total** | **~8-14 GB** | Full pipeline run |

## 🗑️ Cleaning Up

To remove generated files and start fresh:

```bash
# Remove all generated outputs (keeps only raw input files)
cd data/
rm -rf plink_individual/ plink_merged/ plink_qc/
rm -rf liftover/ network_analysis/ deduplicated_analysis/
rm *.csv  # Removes all intermediate CSVs (keeps manifest.csv in subdirs)

# Or remove everything except inputs
cd data/
find . -type f ! -name 'README.md' ! -path './opensnp_genotypes_Ancestry__413files/*' -delete
```

⚠️ **Warning:** The above commands will delete all pipeline outputs. Make sure to back up any results you want to keep!

## 🌍 Alternative Data Location

If you want to keep data on an external drive or network storage:

### Option 1: Symlink
```bash
# Create symlink to external drive
ln -s /mnt/external/opensnp_data data
```

### Option 2: Environment Variable
```bash
# Set data directory location
export PROJECT10_DATA_DIR=/mnt/external/opensnp_data

# Run pipeline
python run_pipeline.py
```

### Option 3: Modify config.py
```python
# In config.py, change:
DATA_DIR = Path("/mnt/external/opensnp_data")
```

## 🔒 Data Privacy & Ethics

**Important Considerations:**

1. **Publicly Available Data:** All OpenSNP data used in this project is voluntarily shared by participants under open licenses (CC0/CC-BY).

2. **No Re-identification Attempts:** This pipeline is for aggregate population analysis only. Do not attempt to:
   - Link genetic data to specific individuals
   - Contact data contributors
   - Use data for purposes beyond academic research

3. **Citation:** Always credit OpenSNP and data contributors in publications.

4. **Sensitive Findings:** If analysis reveals unexpected relationships (e.g., misattributed paternity), treat with appropriate discretion.

## 📋 Data Quality Notes

### Known Issues

- **Build Heterogeneity:** ~15% of files lack build information in headers
- **Format Variants:** Some files use non-standard delimiters (mixed tabs/spaces)
- **Missing Data:** Files have varying amounts of missing genotypes (0-10%)
- **Platform Differences:** 23andMe v3/v4/v5, AncestryDNA v1/v2 have different SNP panels

### Quality Metrics (After Full Pipeline)

| Metric | Initial | After Step 03 | After Step 05 | After Step 07 |
|--------|---------|---------------|---------------|---------------|
| Files | 405 | 329 | 296 | 260 |
| Median SNPs/file | 650k | 680k | 680k | 53k (pruned) |
| Build GRCh37 | 71% | 87% | 100% | 100% |
| Pass rate | - | 81% | 73% | 64% |
